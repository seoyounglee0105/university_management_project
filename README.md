# university_management_project
프로젝트 개요
============
* #### 대학 학사정보 시스템
  * #### 대학교에서 사용하는 학사정보시스템을 CRUD를 중심으로 가볍게 구현하였다.

프로젝트 기간
============
* #### 2023-04-21 ~ 2023-05-02

사용 기술스택
============
* #### BackEnd: JAVA(JDK11), JSP, SpringBoot, MySQL
* #### FrontEnd: HTML5, CSS, JavaScript
* #### collaboration tools: Git, GitHub, Collabee

주요기능
========
* #### 공통기능
  * #### 로그인
    * #### 세션처리
    * #### 쿠키저장
    * #### 아이디 찾기
    * #### 비밀번호 찾기
  * #### 회원정보
    * #### 내 정보 조회
    * #### 내 정보 수정
    * #### 비밀번호 수정
  * #### 공지 및 학사일정
    * #### 공지 조회
    * #### 학사일정 조회
